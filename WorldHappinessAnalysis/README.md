# World Happiness Report Analysis

## Overview

This project explores the factors affecting happiness levels across countries using data from the World Happiness Report.

The analysis focuses on identifying relationships between happiness scores and socioeconomic indicators through data visualization and exploratory data analysis (EDA).

---

## Project Workflow

* Data cleaning and preprocessing
* Missing value analysis
* Exploratory Data Analysis (EDA)
* Correlation analysis
* Regression-based visualizations
* Country and regional comparisons
* Temporal analysis (2007–2023)
* Geographic visualization using world maps

---

## Key Findings

* GDP per Capita, Social Support, and Healthy Life Expectancy show strong positive relationships with happiness.
* Countries with higher freedom of choice tend to report higher happiness levels.
* Significant differences exist between world regions in happiness scores.
* Happiness trends have changed over time across different countries and regions.
