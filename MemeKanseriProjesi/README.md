# Breast Cancer Classification Project

This project focuses on breast cancer histopathology image classification using deep learning.

## Dataset
BreaKHis Dataset

## Methods
- Patient-level data splitting
- Data leakage prevention
- Transfer Learning
- Data Augmentation
- 5-Fold Stratified Group Cross Validation

## Models
- CNN
- EfficientNetB0
- DenseNet121

## Best Model
DenseNet121
